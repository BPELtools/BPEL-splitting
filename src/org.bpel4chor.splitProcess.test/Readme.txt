To run the Test for org.bpel4chor.splitProcess
-----------------------------------------------

1. run test suite e.g. org.bpel4chor.splitprocess.test.AllTest 
(a) open the AllTest.java 
(b) select in menu Run -> Run as -> JUnit Test

2. run single test case e.g. org.bpel4chor.splitprocess.test.utils.ActivityFinderTest.java
(a) open the ActivityFinderTest.java
(b) select in menu Run -> Run as -> JUnit Test

3. run single test e.g. org.bpel4chor.splitprocess.test.utils.ActivityUtilTest.testIsSimpleActivity()
(a) open the class ActivityFinderTest.java
(b) set focus on the method testIsSimpleActivity() in the "outline explorer"
(c) select in menu Run->Run as->JUnit Test
